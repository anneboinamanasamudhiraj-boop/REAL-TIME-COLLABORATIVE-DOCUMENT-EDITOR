# Real-Time Collaborative Document Editor

## Project Overview

This project is a Real-Time Collaborative Document Editor developed using React.js, Node.js, Express.js, and Socket.IO. Multiple users can edit the same document simultaneously, and changes are reflected instantly across all connected clients.

##  Features

* Real-time text synchronization
* Multiple users can edit simultaneously
* Instant updates across browser tabs
* Socket.IO-based communication
* Simple and responsive user interface

## Technologies Used

### Frontend

* React.js
* JavaScript
* HTML
* CSS

### Backend

* Node.js
* Express.js
* Socket.IO

## Installation and Setup

### Clone Repository

git clone <repository-url>

### Install Backend Dependencies

npm install express socket.io cors

### Start Backend Server

cd server

node server.js

### Install Frontend Dependencies

cd client

npm install

npm install socket.io-client

### Start React Application

npm start

##  Usage

1. Start the backend server.
2. Start the React frontend.
3. Open the application in two browser tabs.
4. Type text in one tab.
5. The same content will instantly appear in the other tab.

##  Learning Outcomes

* Understanding WebSocket communication
* Implementing real-time applications
* React and Socket.IO integration
* Client-server architecture

## Conclusion

The Real-Time Collaborative Document Editor demonstrates how Socket.IO can be used to build collaborative applications with instant synchronization between multiple users. The project successfully enables real-time document editing through a simple and efficient architecture.
